L'étrange Bonbon

-Bande dessinée interactive- 2020

L'objectif de ce projet était de réaliser un mini site web qui raconte les début d'internet.
La préface du livre de Fred Turner, "Aux sources de l'utopie numérique" était le document source. Ainsi, quoi de mieux qu'une bande dessinée interactive pour traiter ce sujet.
